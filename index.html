Orientation / keepsake visualization built from real RAAVEN B1 flight data () — not a scientific analysis figure. Position is a flat-earth projection relative to the Marshall Field Site tower (39.94796°N, −105.19576°W); phase coloring (ascent / cruise / descent) is a simple vertical-rate heuristic (dz/dt = −VD, ±0.5 m/s), separate from the project's actual QC/pass classification used in load_b1.m. Data downsampled 10 Hz native → 1 point / 2 s for smooth playback. The 3D Orbit view exaggerates altitude ×5 (labeled) so climb/descent reads clearly against the ~900 m horizontal extent of the flight.
